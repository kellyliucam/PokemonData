# PokéDex 日本語 — Japanese Pokémon Name Lookup

## Features

- **Search by English Name** — Type any Pokémon's English name and get auto-complete suggestions. Selecting a result shows the official Japanese katakana name, Pokédex number, and official image.
- **Search by Pokédex Number** — Enter a  Pokédex number (e.g. `22`) to instantly retrieve the Pokémon's English name, Japanese name, and image.
- **Image display** via the [PokéAPI sprites repository](https://pokeapi.co/)
- **Keyboard navigation** — Arrow keys and Enter work on the suggestion dropdown
- **Covers all generations I–IX** (~1,010 Pokémon)

## Data Source

Pokémon name data is sourced from:

- **Bulbapedia** — [List of Japanese Pokémon names](https://bulbapedia.bulbagarden.net/wiki/List_of_Japanese_Pok%C3%A9mon_names)  
  Bulbapedia is a community-driven Pokémon encyclopedia. Japanese names are the official names as dictated by Nintendo/Game Freak.

- **PokéAPI** — [https://pokeapi.co/](https://pokeapi.co/)  
  Used to retrieve Pokémon images via their GitHub-hosted image assets.
